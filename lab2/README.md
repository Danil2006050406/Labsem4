# Algorithm Visualizer

**Algorithm Visualizer** — це десктопний застосунок, розроблений на **C++ з використанням Qt 6**, призначений для візуалізації та порівняння алгоритмів сортування.

Програма дозволяє користувачу вводити масив чисел, обирати алгоритм сортування та спостерігати процес сортування у вигляді анімованих стовпчиків.

---

## Основні можливості

* візуалізація процесу сортування за допомогою гістограми;
* підтримка кількох алгоритмів сортування:

  * Bubble Sort
  * Quick Sort
  * Merge Sort
* порівняння швидкодії алгоритмів;
* вимірювання часу виконання;
* збереження історії сортування;
* модульне тестування за допомогою Google Test;
* сучасний графічний інтерфейс на Qt Widgets.

---

## Використані шаблони проєктування

У проєкті реалізовано кілька **патернів проєктування**:

* **Strategy** — для вибору алгоритму сортування;
* **Factory** — для створення об’єктів алгоритмів;
* **Facade** — спрощений інтерфейс для запуску сортування;
* **Decorator** — вимірювання часу виконання;
* **Memento** — збереження станів масиву;
* **Command** — виконання операцій обміну елементів.

Це робить проєкт зручним для навчальних та лабораторних робіт з програмної інженерії.
