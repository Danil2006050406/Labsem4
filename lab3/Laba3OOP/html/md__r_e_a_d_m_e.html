Лабораторна робота №3b

Паралельне мультипоточне програмування

Автор

Сахно Данило Студент КНУ ім. Тараса Шевченка Факультет комп'ютерних наук та кібернетики

Опис роботи

У даній лабораторній роботі реалізовано послідовну та паралельну версії алгоритму сортування злиттям (Merge Sort) з використанням технологій мультипоточного програмування.

Метою роботи є:

  • дослідження ефективності паралельних алгоритмів
  • порівняння часу виконання з послідовною реалізацією
  • реалізація зручного графічного інтерфейсу

Реалізовано

  • Послідовний алгоритм Merge Sort
  • Паралельний Merge Sort (з використанням багатопоточності)
  • Графічний інтерфейс користувача (Qt)
  • Порівняння результатів роботи алгоритмів
  • Вимірювання часу виконання
  • Збереження результатів у CSV-файл
  • Виконання алгоритму у фоновому потоці (без блокування GUI)
  • Юніт-тести (Google Test)
  • Документація коду (Doxygen)

Архітектура та ООП

У проєкті використано об’єктно-орієнтований підхід:

  • ISorter — абстрактний інтерфейс сортування
  • SequentialMergeSorter — послідовна реалізація
  • ParallelMergeSorter — паралельна реалізація
  • SortContext — реалізація патерну Strategy

Це дозволяє:

  • легко змінювати алгоритм під час виконання
  • розширювати систему новими алгоритмами

Мультипоточність

Паралельна версія алгоритму використовує:

  • поділ задачі на підзадачі
  • виконання частин сортування у різних потоках

Це дозволяє значно прискорити обробку великих масивів даних.

Тестування

Було реалізовано юніт-тести з використанням Google Test, які перевіряють:

  • коректність сортування
  • обробку порожніх масивів
  • роботу з одним елементом
  • відповідність результатів паралельної та послідовної версій
  • роботу на великих масивах

Результати

Результати автоматично зберігаються у файл performance.csv.

Висновок

Паралельна реалізація алгоритму показує кращу продуктивність при великих обсягах даних завдяки використанню декількох потоків.

Водночас для малих масивів накладні витрати на створення потоків можуть зменшувати ефективність.

Як запустити

  1. Зібрати проєкт через CMake
  2. Запустити програму
  3. Ввести розмір масиву
  4. Натиснути кнопку Run Sort
  5. Переглянути результати
